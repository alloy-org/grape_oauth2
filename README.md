# Grape-OAuth2

This gem adds a flexible OAuth2 authentication for your Grape project.


## Installation

If using bundler, first add 'grape_oauth2' to your Gemfile:

```ruby
gem "grape_oauth2"
```

And run:

```sh
bundle install
```

Otherwise simply install the gem:

```sh
gem install grape_oauth2
```
